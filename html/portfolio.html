Meus Projetos

Tenho alguns projetos que considero bastante interessantes e que me proporcionaram aprendizados valiosos, todos disponíveis no meu perfil do GitHub. Atualmente, estou desenvolvendo o projeto UmbrellaCorporation. Um projeto anterior, criado em Java, tem como objetivo a gestão de farmácias, funcionando como um software interno da empresa. Inspirado por essa experiência, decidi criar outro sistema utilizando o framework React Native para dispositivos móveis e a mesma base de dados do projeto de gerenciamento de farmácia, com TypeScript como linguagem. Inicialmente, contei com a ajuda de dois amigos desenvolvedores, e juntos conseguimos gerir o projeto e distribuir responsabilidades, resultando na primeira build de teste. Agora, estou continuando o trabalho sozinho. No entanto, estou enfrentando dificuldades na implementação do banco de dados, pois requer domínio de PHP para comunicação, uma linguagem que ainda não conheço muito bem.

Outro projeto interessante foi a criação de um helicóptero 3D usando C. Esse projeto foi desafiador devido à tecnologia OpenGL, que é bastante antiga e tem pouco material de estudo disponível. Escrevi muitas linhas de código para criar o helicóptero, utilizando apenas formas geométricas, o que exigiu muitos cálculos e parametrizações de arestas. Além disso, implementei funcionalidades como disparar com metralhadora, lançar mísseis, movimentação básica, movimentação de câmera, zoom-in e zoom-out. Apesar do estresse, o aprendizado e a experiência adquiridos foram extremamente valiosos.

Esses são apenas alguns exemplos dos meus projetos. Estou pensando em revisar esses projetos para uma futura refatoração de código e criação de documentação, algo que já estou fazendo para os projetos mais atuais.
